# Pinecone

Link: https://www.pinecone.io/ (serviço)

Categoria: Vector DB (SaaS)

Resumo
Serviço gerenciado de vector database com foco em simplicidade de uso, escala automática e latência baixa. Não é OSS, mas é amplamente usado em produção.

Por que acompanhar
Mudanças de API e novos features podem facilitar ou exigir adaptações em pipelines de RAG.

Pontos de interesse
- API compatibilidade
- Cost & scaling characteristics

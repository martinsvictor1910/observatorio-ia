Contribuições

Obrigado por considerar contribuir com este Observatório. Objetivo: manter um repositório confiável de recomendações e melhores práticas para soluções de IA na área de Dados.

Como contribuir
- Abra uma issue para sugerir um novo projeto/skill ou correção.
- Para adicionar um novo item: crie um arquivo em `skills/<categoria>/` seguindo o template existente (título, link, categoria, resumo de 1–2 linhas, por que acompanhar, pontos de interesse, tags).
- Faça um fork, crie uma branch com nome `feature/<algo>` e abra um PR.

Padrões
- Conteúdo em português (preferência) ou inglês se o projeto só tiver documentação em inglês.
- Mantenha resumos objetivos (3-8 frases) e links oficiais.

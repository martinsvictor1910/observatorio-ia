# DVC (Data Version Control)

Link: https://github.com/iterative/dvc

Categoria: Versionamento de Dados / Pipelines reprodutíveis

Resumo
Ferramenta para versionar dados e modelos por meio de Git (metadados) e storage remoto para artefatos grandes.

Por que acompanhar
Melhorias em integração CI/CD, suporte a remotes, performance e colaboração em pipelines de dados.

Pontos de interesse
- Pipeline reproducibility
- Storage remotes e cache

Tags: dvc, data-versioning, reproducibility

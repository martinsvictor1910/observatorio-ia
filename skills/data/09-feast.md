# Feast

Link: https://github.com/feast-dev/feast

Categoria: Feature Store / Production ML

Resumo
Feature store open-source para gerenciar features em produção, sincronizar offline/online stores e servir features para modelos em inferência.

Por que acompanhar
Novas integrações com stores (Redis, Databricks), performance e suportes a pipelines de features.

Pontos de interesse
- Store connectors (online/offline)
- Feature materialization e serving

Tags: feature-store, feast, serving

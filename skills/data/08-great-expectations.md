# Great Expectations

Link: https://github.com/great-expectations/great_expectations

Categoria: Data Quality / Testing

Resumo
Framework para validar e monitorar qualidade de dados com expectativas declarativas e integração com pipelines.

Por que acompanhar
Novos conectores, integração com data catalogs e automações de testes em pipelines.

Pontos de interesse
- Checkpoints e monitoração
- Integração com data warehouses e orchestration

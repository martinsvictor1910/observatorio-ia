# Dagster

Link: https://github.com/dagster-io/dagster

Categoria: Orquestração / Data Engineering

Resumo
Framework de orquestração com foco em teste local, tipos e observabilidade; design voltado a desenvolvedores de dados com boas práticas de engenharia.

Por que acompanhar
Evolução das APIs de tipo, integração com ferramentas de observabilidade e melhorias na experiência de desenvolvimento local.

Pontos de interesse
- Solid/Op abstractions
- Integração com data catalogs e feature stores

Tags: dagster, orchestration, observability

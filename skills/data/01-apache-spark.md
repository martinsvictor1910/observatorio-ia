# Apache Spark

Link: https://github.com/apache/spark

Categoria: Data Processing / Big Data / ML infra

Resumo
Motor distribuído para processamento em larga escala (batch e streaming), usado para ETL, análise, ML e processamento de eventos. Padrão de fato em muitos pipelines de dados corporativos.

Por que acompanhar
Mudanças em APIs, otimizações de SQL/Planner, compatibilidade com versões de Scala/Java, integrações com Delta Lake e melhorias em runtime podem impactar pipelines em produção.

Pontos de interesse
- Releases (breaking changes em versões maiores)
- Integração com Databricks e runtimes gerenciados
- Projetos relacionados: Spark SQL, Structured Streaming

Tags: spark, etl, streaming, databricks, delta

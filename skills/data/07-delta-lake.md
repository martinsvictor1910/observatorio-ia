# Delta Lake

Link: https://github.com/delta-io/delta

Categoria: Storage / Data Lake

Resumo
Projeto que adiciona ACID, versionamento e transações a data lakes (parquet) — base para muitos pipelines com Spark/Databricks.

Por que acompanhar
Novas features de performance, schema evolution, compaction e integração com motores de query.

Pontos de interesse
- Time travel e versionamento
- Compaction / performance tuning

Tags: delta, lakehouse, parquet

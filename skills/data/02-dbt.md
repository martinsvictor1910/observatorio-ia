# dbt (Data Build Tool)

Link: https://github.com/dbt-labs/dbt-core

Categoria: Transformations / Analytics Engineering

Resumo
Framework para transformar dados dentro do warehouse usando SQL modular, testes e versionamento. Facilita engenharia analítica com práticas de software (CI, revisão, testes).

Por que acompanhar
Novas features de materializations, performance, integrações com warehouses (Snowflake, BigQuery, Redshift) e melhorias no ecossistema de macros e packages.

Pontos de interesse
- Releases e mudanças em adapters
- Novas práticas de testing e CI/CD

Tags: dbt, sql, analytics

# LangChain

Link: https://github.com/langchain-ai/langchain

Categoria: Agents / LLM orchestration / Retrieval

Resumo
Framework para construir aplicações com LLMs, ferramentas de RAG (retrieval-augmented generation), agentes e fluxos conectando LLMs a fontes de dados.

Por que acompanhar
Novas integrações com vector stores, patterns de agentes, e componentes de chains que facilitam aplicações de IA sobre dados.

Pontos de interesse
- Connectors para vector stores
- Agent frameworks e toolkits

Tags: langchain, agents, rag, retrieval

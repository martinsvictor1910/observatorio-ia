# Auto-GPT

Link: https://github.com/Significant-Gravitas/Auto-GPT

Categoria: Agents / Autonomous Agents (experimentais)

Resumo
Projeto experimental que demonstra loops autônomos de agentes usando LLMs para encadear tarefas. Útil como referência, mas exige cautela para produção.

Por que acompanhar
Evolução de patterns de agentes autônomos, exemplos de prompts e automações que podem inspirar soluções centradas em dados.

Pontos de interesse
- Exemplo de agent loops
- Segurança e controls (rate limits, tool usage)

Tags: autogpt, agents, experimental

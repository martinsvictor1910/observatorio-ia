# BabyAGI

Link: https://github.com/yoheinakajima/babyagi

Categoria: Agents / Experimentation

Resumo
Versão simplificada de agente auto-gerenciável com ciclo de tarefas — útil para estudo e prototipação de pipelines autônomos.

Por que acompanhar
Novas ideias para organização de tasks e memory management em agentes que interajam com dados.

Pontos de interesse
- Task prioritization
- Memory & context management

# n8n

Link: https://github.com/n8n-io/n8n

Categoria: No-code / Automation / Integrations

Resumo
Plataforma de automação de workflows com nodes que conectam APIs e serviços — ideal para prototipar integrações e alertas sem código.

Por que acompanhar
Novos nodes, connectors e exemplos de uso para monitorar releases, criar alertas e acionar testes de integração.

Pontos de interesse
- Nodes para Slack, HTTP, Webhook, GitHub
- Execução self-hosted vs cloud

Tags: n8n, automation, integrations
